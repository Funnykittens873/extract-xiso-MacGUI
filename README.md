# extract-xiso

A command line utility created by [*in*](mailto:in@fishtank.com) to allow the creation, modification, and extraction of XISOs. Currently being maintained and modernized by the [*XboxDev organization*](https://github.com/XboxDev/XboxDev). MacOS GUI version by [*Funnykittens873*](https://github.com/Funnykittens873).

## Features

- Create XISOs from a directory.

- Extract XISO content to a directory.

- Multi-Platform and Open-Source.
